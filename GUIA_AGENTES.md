# 🤖 Guia de Agentes Antigravity

Este guia resume os **20 agentes especialistas** disponíveis para te ajudar neste projeto. Você pode invocar qualquer um deles mencionando o nome (ex: "Peça ao @frontend-specialist para...") ou o @orchestrator escolherá os melhores para tarefas complexas.

---

## 🚀 1. Gestão e Planejamento
Agentes que ajudam a organizar o trabalho antes de começar a codar.

| Agente | Quando Usar |
| :--- | :--- |
| **`orchestrator`** | **O Coordenador Mestre.** Use para tarefas grandes que envolvem várias áreas ao mesmo tempo. |
| **`project-planner`** | Para criar planos de execução, dividir tarefas em fases e definir metas. |
| **`product-manager`** | Para discutir requisitos, funcionalidades de negócio e visão do produto. |
| **`product-owner`** | Focado em priorizar o "Backlog" e garantir que o app entrega valor. |

## 💻 2. Desenvolvimento e Design
Os especialistas que colocam a mão na massa no código e na interface.

| Agente | Quando Usar |
| :--- | :--- |
| **`frontend-specialist`** | Especialista em **UI/UX, React, Next.js e Tailwind.** Foco em telas bonitas e rápidas. |
| **`backend-specialist`** | Especialista em **Server Actions, APIs, Node.js e Lógica de Negócio.** |
| **`database-architect`**| Design de tabelas no **Supabase, Migrações SQL** e otimização de buscas. |
| **`mobile-developer`** | Criar apps para iOS e Android usando React Native ou Flutter. |
| **`game-developer`**   | Lógica de jogos, gerenciamento de cenas e motores como Phaser ou Unity. |

## 🧪 3. Qualidade e Performance
Garantem que o app não quebre e que seja extremamente rápido.

| Agente | Quando Usar |
| :--- | :--- |
| **`qa-automation-engineer`** | Criar fluxos de teste automáticos com **Playwright** (como fizemos agora). |
| **`test-engineer`** | Focado em testes unitários, testes de integração e cobertura de código. |
| **`debugger`** | **O Detetive.** Ótimo para caçar bugs difíceis e encontrar a causa raiz de erros. |
| **`performance-optimizer`** | Otimizar tempo de carregamento, pacotes pesados e Core Web Vitals. |

## 🛡️ 4. Segurança e Infraestrutura
Proteção de dados e publicação em produção.

| Agente | Quando Usar |
| :--- | :--- |
| **`security-auditor`** | Auditoria de segurança, proteção de rotas e revisão de permissões Supabase RLS. |
| **`devops-engineer`** | **Deploy em VPS**, configuração de Nginx, Docker, CI/CD e Vercel. |
| **`penetration-tester`**| Testes de "hacking" ético para ver se o app aguenta ataques reais. |
| **`code-archaeologist`**| Para entender códigos antigos ou arquivos gigantes que ninguém mexeu há muito tempo. |

## ✨ 5. Especialistas Adicionais

| Agente | Quando Usar |
| :--- | :--- |
| **`seo-specialist`** | Otimização para o Google, Meta Tags e Google Search Console. |
| **`documentation-writer`**| Escrever READMEs impecáveis, documentação de API e tutoriais de uso. |
| **`explorer-agent`** | Mapear a estrutura de pastas e dependências de um projeto desconhecido. |
